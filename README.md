# Inventory Dashboard App

A business inventory management application with a dynamic product table and an analytics dashboard.

## Features
- Product CRUD operations
- Bulk selection and deletion
- Real-time inventory analytics
- Visual insights using charts (category distribution, top products)
- Authenticated user support

## Tech Stack
- **Frontend:** React, MUI, MUI X Data Grid, MUI X Charts
- **Backend:** Node.js, Express
- **Database:** PostgreSQL + Prisma
- **Auth:** JWT (access + refresh tokens)
